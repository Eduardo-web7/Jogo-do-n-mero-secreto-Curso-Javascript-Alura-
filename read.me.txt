# 🔐 Jogo de Número Secreto

Projeto desenvolvido durante o curso **"Lógica de Programação: Mergulhe em JavaScript"** da [Alura](https://www.alura.com.br). O objetivo é aplicar conceitos básicos de programação em um jogo simples e interativo.

## 🧩 Sobre o jogo

O sistema gera um número aleatório e o jogador deve tentar adivinhar. A cada tentativa, o jogo fornece dicas se o número é maior ou menor. Ao acertar, o jogo reinicia automaticamente.

## 🛠️ Tecnologias utilizadas

- JavaScript
- HTML
- CSS (opcional para estilização)

## 📚 Conceitos aplicados

- Variáveis e tipos de dados
- Condicionais (`if`, `else`)
- Laços de repetição (`while`)
- Funções
- Interação com o usuário via `prompt` e `alert`
- **Operador ternário**: utilizado para simplificar estruturas condicionais.
  
## 🚀 Como executar

1. Clone o repositório  
2. Abra o arquivo `.html` em seu navegador  
3. Siga as instruções na tela e tente descobrir o número secreto!

## 💡 Melhorias futuras

- Interface gráfica com HTML/CSS
- Modo difícil com limite de tentativas
- Histórico de partidas e pontuação
